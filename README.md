# House-Price-Prediction-ML

#  House Price Prediction using Machine Learning

##  Overview
This project predicts house prices using Linear Regression based on features like area, bedrooms, and location.

---

##  Steps Performed
- Data Loading & Exploration
- Feature Engineering (Date → Year, Month, Day)
- Null Value Treatment
- Outlier Removal (IQR Method)
- Data Visualization
- Model Training (Linear Regression)
- Model Evaluation (MAE, RMSE, R² Score)

---

##  Key Insights
- Larger living area → higher price
- Data had outliers which were removed using IQR
- Model achieved decent R² score

---

##  Model Used
- Linear Regression

---

## 📈 Evaluation Metrics
- MAE
- MSE
- RMSE
- R² Score
